# renpynovella
# Первый день в Японии — Ren'Py concept v0.1

Это первый рабочий концепт визуальной новеллы на Ren'Py.


- Играбельная Глава 1 «Первый день».
- Сцены:
  - вокзал и растерянность Петра;
  - встреча с Акари;
  - первое приветствие `こんにちは / konnichiwa`;
  - благодарность `ありがとうございます / arigatou gozaimasu`;
  - извинение `すみません / sumimasen`;
  - короткая викторина в автобусе;
  - появление Сэнсэя.
- Система очков:
  - `jp_score` — правильные учебные ответы;
  - `jp_total` — всего учебных проверок;
  - `confidence` — уверенность Петра;
  - `akari_trust` — отношение Акари.
- Временные заглушки вместо фонов и спрайтов, чтобы проект запускался без ассетов.

## Как запустить

1. Установите Ren'Py.
2. Распакуйте архив.
3. Положите папку `renpy_jp_novel_concept` туда, где Ren'Py видит проекты.
4. Откройте Ren'Py Launcher.
5. Выберите проект `renpy_jp_novel_concept`.
6. Нажмите `Launch Project`.
